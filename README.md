# TP2 JEE Project

This is a Java EE project for learning and practicing web development concepts.

## Requirements
**JDK 8 or higher** <br>
**Maven** for dependency management and project build<br>
**JEE-compatible server** (such as Tomcat, WildFly, or GlassFish) <br>

## Technologies Used
- Java EE
- Servlets
- JSP
- Tomcat Server
